# My Portfolio

A simple, responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Features

- Responsive design that works on desktop and mobile
- Smooth scrolling navigation
- About section with photo placeholder
- Projects showcase grid
- Contact form with basic validation
- Clean, modern styling

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Customize the content to match your information

## Customization

- **Personal Info**: Edit the text in `index.html` to replace placeholders like [Your Name], [Your Profession], etc.
- **Photo**: Replace `placeholder.jpg` with your actual photo (200x200px recommended)
- **Projects**: Update the project cards in the HTML with your actual projects
- **Styling**: Modify `styles.css` to change colors, fonts, or layout
- **Functionality**: Add more JavaScript features in `script.js`

## File Structure

```
portfolio/
├── index.html      # Main HTML file
├── styles.css      # CSS styles
├── script.js       # JavaScript functionality
├── placeholder.jpg # Photo placeholder (replace with your image)
└── README.md       # This file
```

## Browser Support

Works in all modern browsers including Chrome, Firefox, Safari, and Edge.

## Troubleshooting

- If images don't load, check the file paths
- For mobile issues, test in browser dev tools responsive mode
- Contact form currently shows an alert; integrate with a backend service for real email sending
